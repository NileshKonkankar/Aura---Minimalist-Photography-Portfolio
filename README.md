# Aura - Minimalist Photography Portfolio

Aura is a minimalist, high-performance portfolio template designed for professional photographers to showcase their creative work. Built with modern web technologies, it provides an elegant, distraction-free environment where your photography takes center stage.

## ✨ Features

*   **Immersive Hero Section:** Animated parallax background with staggered typography and a subtle film-grain texture.
*   **Masonry Portfolio Grid:** Beautifully displays photos of varying aspect ratios (portrait, landscape, square) without cropping.
*   **Infinite Scrolling:** Seamlessly loads more images as the user scrolls down the page.
*   **Cinematic Lightbox:** A premium, full-screen image viewer with blur transitions and scale effects.
*   **Interactive Background Effects:** Fluid ambient light orbs that react to mouse movement and scroll position.
*   **Category Filtering:** Easily filter portfolio items by categories like Portrait, Architecture, and Nature.
*   **Fully Responsive:** Flawless experience across mobile, tablet, and desktop devices.

## 🛠️ Tech Stack

*   **Framework:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
*   **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
*   **Animations:** [Framer Motion](https://www.framer.com/motion/)
*   **Icons:** [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/aura-portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd aura-portfolio
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## 🎨 Customization

*   **Adding Photos:** Open `App.tsx` and update the `PORTFOLIO` array with your own image URLs, titles, and categories.
*   **Changing the Name:** Update the "Alex Morgan" placeholder text in `App.tsx` and `index.html` to your own name.
*   **Categories:** Modify the `CATEGORIES` array in `App.tsx` to match your photography niches.
